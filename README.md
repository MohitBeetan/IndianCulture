# Bharat — The Soul of India

A beautifully designed, responsive website celebrating the rich cultural heritage, history, and traditions of India. This project showcases Indian culture through interactive sections covering history, traditions, architecture, festivals, cuisine, and more.

## 🌍 Overview

This is a single-page application with multiple sections that takes users on a journey through Indian culture, art, architecture, and lifestyle. The website features a warm, Indian-inspired visual aesthetic with smooth animations and responsive design.

## ✨ Features

- **Multi-section layout**: History, Culture, Architecture, Festivals, Cuisine, States, and Philosophy
- **Responsive design**: Fully optimized for desktop, tablet, and mobile devices
- **Interactive navigation**: Smooth scrolling with page indicators and navigation menu
- **Custom styling**: Indian-inspired color palette with warm tones and elegant typography
- **Animated elements**: Including custom mandala SVG graphics and interactive cursors
- **Mobile-friendly navigation**: Toggle navigation menu for smaller screens

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- A modern web browser

### Installation

1. Clone the repository or navigate to the project directory:
```bash
cd FrontendBackendProject
```

2. Install dependencies:
```bash
npm install
```

3. Start the server (once `scerver.js` is configured):
```bash
node scerver.js
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

Or simply open `public/index.html` directly in your browser for a quick preview.

## 📖 Navigation Sections

- **History**: Explore India's historical timeline and significant events
- **Culture**: Discover the diverse cultural traditions across India
- **Architecture**: Learn about iconic architectural monuments and their significance
- **Festivals**: Experience the vibrant festivals celebrated throughout India
- **Cuisine**: Journey through regional Indian culinary traditions
- **States**: Learn about different states and their unique characteristics
- **Philosophy**: Explore spiritual and philosophical teachings

## 🎨 Design Features

- **Color Palette**: Warm Indian-inspired colors including golds, terracottas, and deep jewel tones
- **Typography**: Uses decorative fonts (Yatra One, Cinzel) alongside readable serif fonts (Lora)
- **Mandala Graphics**: SVG-based mandala patterns for cultural authenticity
- **Smooth Animations**: Enhanced user experience with subtle transitions and scrolling effects
- **Custom Cursor**: Interactive cursor element following mouse movement

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js (to be configured in `scerver.js`)
- **Server**: Express.js (recommended for `scerver.js`)

## 📝 File Descriptions

| File | Purpose |
|------|---------|
| **index.html** | Main page with hero section, navigation, and all content sections |
| **style.css** | All styling including responsive design and animations |
| **script.js** | JavaScript functionality for interactive elements and page navigation |
| **scerver.js** | Node.js server file - can be used to serve static files and handle backend logic |

## 📱 Responsive Breakpoints

The website is optimized for:
- **Desktop**: 1024px and above
- **Tablet**: 768px to 1023px
- **Mobile**: Below 768px
